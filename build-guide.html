Build Your APK

Once Gradle sync is complete, you're ready to build!

Build steps:

  • Click "Build" in the top menu
  • Select "Build Bundle(s) / APK(s)"
  • Click "Build APK(s)"
  • Wait 2-3 minutes for the build to complete
  • Click "locate" in the notification that appears

Troubleshooting

Gradle Sync Failed?

  • Go to Tools ‚Üí SDK Manager
  • Install "Android SDK Platform 34"
  • Install "Android SDK Build-Tools 34.0.0"
  • Click "File ‚Üí Sync Project with Gradle Files"

Build Failed?

  • Click "Build ‚Üí Clean Project"
  • Wait for it to finish
  • Click "Build ‚Üí Rebuild Project"

Can't install APK on phone?

  • Go to Settings ‚Üí Security
  • Enable "Unknown sources" or "Install unknown apps"
  • Try installing again
